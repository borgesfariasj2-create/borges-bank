# 💰 Borges Bank (Console Banking System)

A simple banking system developed in Java, running entirely in the terminal.
This project simulates basic banking operations such as account registration, login, deposits, withdrawals, transfers, and balance checking.

---

## 🚀 Features

* 🔐 User registration (name, CPF, email, password)
* 🔑 Login system with validation
* 💵 Deposit money
* 💸 Withdraw money with balance validation
* 🔄 Transfer between accounts
* 📊 View account balance
* 🔁 Return to main menu system

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/YOUR-USERNAME/borges-bank.git
```

2. Open the project in IntelliJ IDEA

3. Run the `Main.java` file

## 💡 How It Works

### Main Menu

```
1 - Login
2 - Register
3 - Exit
```

### Account Menu

```
1 - Deposit

3 - Transfer
4 - See balance
```

## ⚠️ Notes

* This project uses in-memory storage (ArrayList), meaning data is lost when the program is closed.
* No database integration yet.
* No encryption for passwords (educational purposes only).
